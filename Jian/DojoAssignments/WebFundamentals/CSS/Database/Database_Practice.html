What are Databases?

A structured set of data


Database are a collection of information, they are used for creating, manipulating, and deleting data. There are many different types of databases and they can be classified in a basic approach of relational database or a non-relational database. Relationships means a set of our data is related to another section of our data. Our database are put into sections od data called schemas, inside each schema each data is organized into tables contain columns and rows to house that particular set of data.
